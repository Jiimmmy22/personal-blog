动效的核心不是炫技，而是解释变化。页面为什么出现、卡片为什么进入视野、导航为什么切换，这些都可以通过轻微、短暂、稳定的运动变得更清楚。

## 少即是多

优秀的动效通常很短。它不抢内容的注意力，只是在用户即将困惑的一瞬间补上一点方向感。

```
initial={{ opacity: 0, y: 18 }}
animate={{ opacity: 1, y: 0 }}
```

这类微小位移比夸张的缩放更耐看，也更接近 Apple、Linear 这类产品的气质。

## 页面切换

App Router 中可以通过 `template.tsx` 包裹一个客户端过渡组件，让每次路由变化都有统一的进入和离开动画。
